Everything you need

A complete AI coding toolkit, designed for developers who want control and flexibility.

⚙

Model Agnostic

Use Anthropic, OpenAI, Ollama, or any OpenAI-compatible API. Switch providers with a single flag — your workflow stays the same.

✎

10 Built-in Tools

Read, write, edit files. Run bash commands. Search with glob and grep. Diff files. Fetch URLs. All with structured, typed outputs.

☌

Safety & Permissions

Filesystem sandbox, command blocklists, rate limiting, audit trails, and undo/rollback. Full control over what the agent can do.

☁

Plugin System

Extend with plugins, MCP servers, and custom hooks. Register new tools dynamically. Add pre/post middleware to any tool execution.

⇄

Session Management

Auto-save conversations, resume later, export as markdown or HTML. Automatic compaction keeps context windows efficient.

⚡

Built with Rust

Fast, reliable, and memory-safe. Compiles to a single binary. No runtime dependencies, no Docker, no Node.js required.

Clean, layered architecture

A modular Cargo workspace with clear separation of concerns, making it easy to extend and contribute.
