10 Built-in Tools

Read, write, edit files. Run bash commands. Search with glob and grep. Diff files. Fetch URLs. All with structured outputs.

Safety & Permissions

Filesystem sandbox, command blocklists, rate limiting, audit trails, and undo/rollback. Full control over the agent.

Plugin System

Extend with plugins, MCP servers, and custom hooks. Register new tools dynamically with pre/post middleware.

Session Management

Auto-save conversations, resume later, export as markdown or HTML. Automatic compaction for context windows.
